# Сессия: IOS-08 scope — "Моя семья и друзья" (contacts/chips)

Дата: 2026-02-28

## Запрос
В настройках добавить вкладку "Моя семья и друзья", где пользователь управляет списком контактов для запросов рецептов.

- `docs/roadmap.md`
- `docs/current_state.md`

## Зафиксированный scope
- `IOS-08`: Settings -> "Моя семья и друзья" (CRUD контактов, минимум: имя).
- Контакты используются как чипы в экране "Запросить рецепт".
- Тап по чипу подставляет получателя в форму запроса.
- Чипы цветные; цвет контакта стабильный и повторяемый.
- Android parity зафиксирован как `ANDROID-SPEC-04` / `ANDROID-UI-08`.
- Backend-часть вынесена в отдельную спецификацию `BE-SPEC-02`.

## UX-мотивация
- Быстрый выбор получателя без ручного ввода.
- Один централизованный список близких для повторяющихся запросов.
