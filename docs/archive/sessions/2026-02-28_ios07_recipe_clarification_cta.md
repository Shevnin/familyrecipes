# Сессия: IOS-07 scope — "Попросить уточнить" на карточке рецепта

Дата: 2026-02-28

## Запрос
Добавить на карточку рецепта кнопку, через которую можно отправить автору рецепта запрос на уточнение/подредактирование.

## Что обновлено
- `docs/backlog.md`
- `docs/roadmap.md`
- `docs/current_state.md`
- `docs/decisions.md`

## Зафиксированный scope
- `IOS-07`: CTA "Попросить уточнить" на Recipe Detail.
- Кнопка открывает prefilled create-request flow:
  - `recipient_name` = `author_name` рецепта;
  - `dish_name` = `Уточнение: <recipe_title>`.
- Отправка дальше использует существующий share flow.
- Для Android зафиксирован parity: `ANDROID-SPEC-03` / `ANDROID-UI-07`.

## Зачем
- Снимает трение, когда в рецепте есть неясности.
- Быстрый loop "вопрос -> уточнение" прямо из контекста конкретного рецепта.
