# Archive

Здесь лежат материалы, которые больше не участвуют в текущей delivery-работе, но сохраняются как история.

## Содержимое
- `prototype_flows/` - ранние flow-документы времен прототипа.
- `sessions/` - завершенные и больше не активные session-логи.
- `design/` - устаревшие дизайн-брифы, мокапы и промежуточные артефакты.
- `legacy/project_legacy.md` - прежний файл проекта до упрощения структуры docs.
- `legacy/user_tasks_legacy.md` - ранний каталог пользовательских задач.

## Правило
- В `docs/` (корень) держим только актуальные рабочие документы.
- В `docs/sessions/` держим только последние опорные сессии и активные task packs.
- Все устаревшее переносим в `archive/`, а не удаляем безвозвратно.
