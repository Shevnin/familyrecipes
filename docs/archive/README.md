# Archive

Здесь лежат материалы, которые не участвуют в текущем discovery-цикле, но сохраняются как история.

## Содержимое
- `prototype_flows/` - ранние flow-документы времен прототипа.
- `legacy/project_legacy.md` - прежний файл проекта до упрощения структуры docs.
- `legacy/user_tasks_legacy.md` - ранний каталог пользовательских задач.

## Правило
- В `docs/` (корень) держим только актуальные рабочие документы.
- Все устаревшее переносим в `archive/`, а не удаляем безвозвратно.
