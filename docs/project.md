# FamilyRecipes

Проект: **FamilyRecipes** — приложение для обмена рецептами.

## Как работаем
Итеративно: **прототип → тест флоу → дизайн → спека → код (Claude Code)**.

Роли:
- **ChatGPT** = Product/UX + документация: user flows, экраны/wireframes, тексты, acceptance criteria, decision log, задания для Claude Code.
- **Claude Code** = инженер: делает только по контракту, маленькие изменения/PR, не рефакторит без запроса.

## Принципы итераций
- Каждая итерация = **1 end-to-end user flow**, который можно пройти в прототипе/приложении.
- Документация — **source of truth**: `/docs/PROJECT.md` + `/docs/DECISIONS.md`.
- Стиль документации: коротко, структурно, без лишних вопросов; если неопределённость — 2 варианта + default.

## Текущий фокус
**Core Flow v1**: открыть рецепт по ссылке → посмотреть → сохранить → найти в “Мои рецепты” → поделиться.

## Репозиторий
- `src/` — прототип (plain HTML/JS)
- `data/` — мок-данные
- `docs/` — продуктовая документация (source of truth)
