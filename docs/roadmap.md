# Roadmap (Discovery Baseline)

Статус: draft, будет пересобираться по итогам discovery-сессий.

## R0 - Problem/Solution Fit (текущий)
Цель: выбрать один главный пользовательский сценарий и подтвердить ценность.

Инициативы:
- Сформировать и согласовать `idea canvas` (`/Users/user/git-project/familyrecipes/docs/templates/idea_canvas.md`).
- Уточнить и закрепить `product brief`.
- Провести серию обсуждений и решений.
- Сформировать приоритизированный backlog.

Критерий выхода:
- Есть согласованный top flow и понятные критерии успеха.

## R1 - Scope for Build
Цель: перевести discovery в реалистичный набор для первой разработки.

Инициативы:
- Определить MVP scope по top flow.
- Подготовить task-card пакет для delivery.
- Зафиксировать ограничения первой итерации.

Критерий выхода:
- Можно отдавать реализацию Claude по маленьким задачам.

## R2 - First Delivery Loop
Цель: реализовать и проверить первый рабочий сценарий.

Инициативы:
- Реализация по task-card.
- Внутренняя проверка against acceptance criteria.
- Коррекция roadmap/backlog по результатам.
