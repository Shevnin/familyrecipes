# Discovery Docs

Папка для материалов product discovery.

## Что здесь хранится
- `idea_canvas.md`: единая рамка идеи и ценности.
- `session_template.md`: шаблон обсуждения идеи.

## Как использовать
1. Начинаем с `idea_canvas.md`.
2. Каждое обсуждение фиксируем отдельным файлом в `/Users/user/git-project/familyrecipes/docs/sessions/` на базе шаблона.
3. После каждой сессии обновляем roadmap, backlog и decisions.
