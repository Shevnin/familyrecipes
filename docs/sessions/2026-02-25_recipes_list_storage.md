# Сессия 2026-02-25 — Recipes List Storage

## Задача
**recipes_list_storage** — список рецептов на экране Мастера загружает данные из localStorage (рецепты, полученные по ссылкам) и объединяет с демо-рецептами.

## Что сделали
- `recipe_responses` из localStorage конвертируются в рецепты и показываются в списке вместе с демо-данными.
- Каждый рецепт из ссылки привязан к запросу через `token` → показывается автор (`target_person`).
- Карточка рецепта: title, sender (с цветовым бейджем), превью, дата, бейдж "по ссылке".
- Детальный просмотр: оригинал + структурированная версия (существующий парсер), бейдж источника, дата.
- Фильтры динамически обновляются — новые отправители появляются автоматически.
- Демо-рецепты получили `source: 'demo'`, ответы по ссылке `source: 'link'`, отправленные мамой `source: 'manual'`.
- ID рецептов: string-based для localStorage рецептов (`resp-<token>`), number для демо — click handler обновлён.
