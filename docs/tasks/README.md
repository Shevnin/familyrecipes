# Tasks Folder

Эта папка хранит task-card'ы для реализации через Claude.

## Как использовать
1. Скопировать шаблон: `/Users/user/git-project/familyrecipes/docs/tasks/_template.md`.
2. Создать новую карточку: `YYYY-MM-DD_<slug>.md`.
3. Заполнить `Goal`, `In scope`, `Out of scope`, `Acceptance criteria`, `Manual test steps`.
4. Только после этого отдавать задачу в реализацию.

## Принципы
- Одна карточка = одна задача = один PR.
- Если scope вырос, создается новая карточка или обновляется текущая до начала кодинга.
- Карточка должна быть самодостаточной: без скрытых договоренностей "в чате".
